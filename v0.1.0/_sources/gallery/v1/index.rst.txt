Available datasets (v1)
=======================

Use the functions in the :mod:`ensaio.v1` module to download and cache (store)
each dataset on your computer.
See the :ref:`documentation of individual functions <api>` for more information
about the respective data, the original data sources, and their licenses.

Information about the data compilation:

:Source: `doi.org/10.5281/zenodo.5167357 <https://doi.org/10.5281/zenodo.5167357>`__
:License: `CC-BY <https://creativecommons.org/licenses/by/4.0/>`__
:Provenance: `github.com/fatiando/data/releases/tag/v1.0.0 <https://github.com/fatiando/data/releases/tag/v1.0.0>`__

These are the datasets that make up the version 1 series:

.. margin::

    Click on the images for examples of fetching, loading, and plotting each
    dataset.
