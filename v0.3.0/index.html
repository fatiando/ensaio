Ensaio

Practice datasets to probe your code

Ensaio (Portuguese for “rehearsal”) is a Python package for downloading open-access sample datasets for Geoscience. It taps into the curated collection of Fatiando a Terra Datasets that is designed for use in tutorials, documentation, and teaching.
